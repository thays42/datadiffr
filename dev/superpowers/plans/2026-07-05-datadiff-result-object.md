# `datadiff_result` Consolidated Comparison Object — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make `compare_data()` return a single, predictable `datadiff_result` object for all outcomes (identical / schema differences / value differences), so `diffdata()` stops overloading its return value to signal a failure.

**Architecture:** A thin S3 record class `datadiff_result` with a `kind` tag and two typed slots (`columns`, `rows`). `compare_data()` gains a column gate that returns `kind = "schema"` instead of aborting; the existing `datadiff_diff` object becomes the `rows` payload, unchanged. `print` / `summary` / `render_diff` gain `datadiff_result` methods that dispatch on `kind`.

**Tech Stack:** R, S3 classes, testthat, roxygen2, cli, checkmate.

## Global Constraints

- Package name is **datadiffr** (directory is `datadiff`); `system.file(..., package = "datadiffr")`.
- Follow tidyverse style: snake_case, `|>` pipe, `stopifnot`/checkmate validation, `cli::cli_abort`/`cli::cli_alert_*` for messaging. Run `make format` before committing.

- All work on branch `feature/datadiff-result-object`.
- S3 method exports are generated by roxygen `#' @export` + `make document`; never hand-edit `NAMESPACE`.

```r
# The consolidated object returned by compare_data(). Exactly one facet is
# populated, keyed by `kind`: "schema" fills `columns` (a compare_columns
# tibble), "value"/"identical" fill `rows` (a datadiff_diff; empty for
# "identical"). `by` and `tolerance` record how the comparison was run.
new_datadiff_result <- function(
  kind,
  columns = NULL,
  rows = NULL,
  by = NULL,
  tolerance = NULL
) {
  structure(
    list(
      kind = kind,
      columns = columns,
      rows = rows,
      by = by,
      tolerance = tolerance
    ),
    class = "datadiff_result"
  )
}
```

```r
#' @export
print.datadiff_result <- function(x, ...) {
  switch(
    x$kind,
    identical = cli::cli_alert_success("No differences found."),
    schema = {
      cli::cli_alert_info("Columns differ; cannot compare values.")
      print(x$columns, ...)
    },
    value = print(x$rows, ...)
  )
  invisible(x)
}

#' @export
summary.datadiff_result <- function(object, ...) {
  structure(
    list(
      kind = object$kind,
      columns = object$columns,
      rows = if (object$kind %in% c("value", "identical")) {
        summary(object$rows)
      } else {
        NULL
      },
      by = object$by,
      tolerance = object$tolerance
    ),
    class = "summary.datadiff_result"
  )
}

#' @export
print.summary.datadiff_result <- function(x, ...) {
  switch(
    x$kind,
    identical = cli::cli_alert_success("No differences found."),
    schema = {
      cli::cli_alert_info(
        "Columns differ ({nrow(x$columns)} column difference{?s})."
      )
      print(x$columns)
    },
    value = print(x$rows)
  )
  invisible(x)
}
```

```r
#' @rdname render_diff
#' @export
render_diff.datadiff_result <- function(diff, output_file = NULL) {
  switch(
    diff$kind,
    identical = {
      cli::cli_alert_success("No differences found.")
      invisible(NULL)
    },
    schema = {
      cli::cli_alert_info("Columns differ; showing schema differences.")
      print(diff$columns)
      invisible(NULL)
    },
    value = render_diff(diff$rows, output_file = output_file)
  )
}
```
